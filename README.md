# Jogo da Velha
Um jogo da velha simples para fins de estudo e aprendizado.

## Como Jogar
É um jogo de estratégia para dois jogadores, que utilizam X ou O para preencher um tabuleiro 3x3.

O objetivo é formar uma linha, coluna ou diagonal com três símbolos iguais, enquanto bloqueia o adversário. O jogo termina quando alguém vence ou o tabuleiro lota e é considerado empate/velha.

## Funcionalidades
- Placar: Um placar que marca a pontuação do jogador em tempo real.

- Sorteio: Por trás da lógica do jogo envolve um algoritmo de sorteio, onde é sorteado quem irá começar a jogar.

- Botão de Reiniciar Jogo: Botão para reiniciar a partida, mantendo o placar dos jogadores intacto e acumulado com os pontos obtidos.
